Detailed Description

template<typename KT, typename VT>
class derecho::cascade::ICascadeObject< KT, VT >

The VT template type of PersistentCascadeStore/VolatileCascadeStore must implement ICascadeObject interface. We use both the concepts of null and valid object in Cascade. A null object precisely means 'no data'; while a valid object literarily means an object is 'valid'. Technically, a null object has a valid key while invalid object does not.
