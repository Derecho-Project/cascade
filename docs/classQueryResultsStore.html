Detailed Description

template<typename T, typename K>
class QueryResultsStore< T, K >

Object made to handle the results of a derecho::rpc::QueryResults object for the C# side. T being the type that is to be returned from the QueryResults object and K being the type that needs to be returned from the lambda unwrapping.

Object made to handle the results of a derecho::rpc::QueryResults object for the python side. T being the type that is to be returned from the QueryResults object and K being the type that needs to be returned from the lambda unwrapping.
